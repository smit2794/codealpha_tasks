# Task 1: Wireframing (Low-Fidelity Design)

This task includes the low-fidelity wireframes of a Food Delivery Mobile Application designed using Figma.

The goal of this task was to create the basic layout and structure of the application before applying visual design. The wireframes focus on user flow and screen arrangement.
